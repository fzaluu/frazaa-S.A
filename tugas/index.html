APA ITU PPLG?

PPLG (Pengembangan Perangkat Lunak dan Gim) adalah sebuah jurusan atau program keahlian yang fokus pada pengembangan perangkat lunak (software) dan pembuatan game. Jurusan ini biasanya terdapat di Sekolah Menengah Kejuruan (SMK) atau lembaga pendidikan vokasi lainnya. PPLG mempelajari berbagai aspek terkait pemrograman, pengembangan aplikasi, desain game, manajemen basis data, dan teknologi informasi secara umum.

Materi yang Dipelajari di Jurusan PPLG

  1. Pemrograman Dasar: Belajar bahasa pemrograman seperti Python, Java, C++, atau JavaScript.
  2. Pengembangan Web: Membuat website menggunakan HTML, CSS, JavaScript, dan framework seperti Bootstrap atau React.
  3. Pengembangan Aplikasi: Membuat aplikasi desktop atau mobile menggunakan tools seperti Android Studio, Flutter, atau Xcode.

Peluang Kerja Jurusan PPLG

  1. Software Developer: Mengembangkan aplikasi atau software untuk berbagai platform.
  2. Pengembangan Web: Membuat website menggunakan HTML, CSS, JavaScript, dan framework seperti Bootstrap atau React.
  3. Pengembangan Aplikasi: Membuat aplikasi desktop atau mobile menggunakan tools seperti Android Studio, Flutter, atau Xcode.

Penghasilan untuk Pemula di Bidang PPLG

  1. Software Developer: Rp 4 - 7 juta per bulan.
  2. Web Developer: Rp 3,5 - 6 juta per bulan.
  3. Game Developer: Rp 4 - 8 juta per bulan
